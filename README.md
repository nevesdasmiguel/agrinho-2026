# agrinho-2026
🌱 Agrinho 2026 – Agro Forte e Futuro Sustentável

Projeto desenvolvido para o Programa Agrinho 2026, com o tema "Agro Forte e Futuro Sustentável". O site apresenta informações sobre a importância do agronegócio, da sustentabilidade, da tecnologia e da educação para o desenvolvimento do campo e da sociedade.

📖 Sobre o Projeto

O objetivo deste trabalho é demonstrar como o agronegócio brasileiro está conectado à inovação, à preservação ambiental e à produção de alimentos que abastecem o país e o mundo.

O site possui um menu lateral para facilitar a navegação entre as seções e utiliza um design simples, organizado e agradável.

🚀 Funcionalidades
Menu lateral de navegação
Seção sobre o Programa Agrinho
Informações sobre o agronegócio brasileiro
